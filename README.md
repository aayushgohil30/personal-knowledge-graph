# personal-knowledge-graph
an inoative way to remember the things which we can't remember 


Personal Knowledge Graph Builder is a browser-based tool that transforms your unstructured notes (PDFs, Word documents, text files) into an interactive visual network. It helps you:

Visualize connections between topics in your notes

Identify knowledge gaps where you have insufficient information

Discover hidden relationships between concepts

Track your learning progress over time

Perfect for students, researchers, lifelong learners, and knowledge workers.
